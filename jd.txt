Agent 应用开发者生态 BD
10-30k·15薪


职位介绍
岗位职责
1. 深入 AI agent 开发者活跃的各类场景（开源社区、技术群、AI 沙龙、黑客松、创业比赛、孵化器与投资人网络等），持续建联优质开发者与团队。
2. 将合适的开发者 / 团队引入公司端侧 AI 生态，协助其在我们的硬件平台上完成 agent 的落地与运行。
3. 从大量建联中识别并筛选出真正具备商业拉力和产品化潜力的精品团队，推动深度合作。
4. 收集一线开发者与市场的真实需求、趋势与痛点，反哺公司产品与场景策略。
5. 在 agent 开发者社区中持续建设公司的技术声量与口碑，吸引优质开发者主动连接。

任职要求
1. 对 AI / agent 有强烈兴趣，具备出色的自我学习能力，遇到陌生领域（含客户所在行业）能主动钻研。
2. 熟悉并已融入 AI / 开发者社区，拥有自己的连接、影响力或作品（如开源项目、技术内容、社群、活动等）。
3. 优秀的沟通与倾听能力，能快速理解对方真实需求，建立信任关系。
4. 具备独立判断力与诚信，能客观评估合作价值，敢于做出取舍。
5. 做事靠谱、说到做到，认同长期主义，愿意陪伴生态共同成长。

加分项
1. 自身有 agent / AI 应用开发经验，能与开发者深度对话；
2. 在某一垂直行业具备理解或资源；
3. 具备内容创作或社区运营能力（能写、能讲、能发声）。

说明：本岗位看重你与 AI 开发者世界的真实连接度与判断力，不以年龄、资历、院校或大厂背景为筛选标准。
